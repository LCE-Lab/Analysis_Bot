# Analysis_Bot
Discord Analysis Bot (Undone)

# Info
Analyze users online, offline, AFK in voice channels.

# How to use
```
1. Edit config.json.example and rename it config.json.
2. npm install
3. node dist
```
Web on `localhost` and port is `8080`.
